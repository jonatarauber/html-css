Tags HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • ParagráfosUse a tag <p>
  • Tutúlos Use as tags <h1> ao <h6>
  • Imagens Use a tag <img>
  • Abreviações Use a tag <abbr>
  • Colocar um link Use a tag <a>
  • Listas númeradas Use as tags <ol> <li>
  • Listas com demarcação Use as tags <ul> <li>
